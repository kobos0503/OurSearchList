<채널 관련 기능>
1. 유튜브 채널을 폴더별로 즐겨찾기 할 수 있다.
2. 유튜브 검색시 즐겨찾기 해놓은 채널이 검색목록 최상단에 위치한다.
3. 즐겨찾기 해놓은 채널 목록을 다른 사람과 공유하고, 손쉽게 내 즐겨찾기로 가져 올수 있는 공유 게시판 글쓰기를 사용할 수 있다.
